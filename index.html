MCP Integration

Add to your claude_desktop_config.json or .claude/settings.json and your agent gets 15 task management tools instantly.

settings.json
{
  "mcpServers": {
    "agentboard": {
      "command": "npx",
      "args": ["-y", "agentboard", "mcp"]
    }
  }
}
